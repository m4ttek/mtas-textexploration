Getting started

Prebuilt jar libraries are available from the download page, current version is 8.1.1.0 (development)

Build from Source with Maven

Download the source code from GitHub

git clone git://github.com/textexploration/mtas.git

Build the library from the created project directory mtas

mvn package

After a successful build, the directory target will contain the new jar library.

Installation

Mtas can be used as plugin for Apache Solr or as library in combination with Apache Lucene.

  • Getting started with Mtas and Lucene
  • Getting started with Mtas and Solr

or see Mtas and Docker to get a quick demonstration.

Versioning follows the required version of both Solr and Lucene followed by an incremental Mtas versioning number. The latest available version is 8.1.1.0, requiring Solr/Lucene version 8.1.1.
